## 🔧 CUSTOMIZATION

### Add Products
**File**: `js/product.js`
```javascript
{
  id: 'product-1',
  name: 'Product Name',
  price: 500000,
  img: '../assets/img/product.jpg',
  category: 'fashion',
  rating: 4.8
}
```

### Change Colors
**File**: `css/theme.css`
```css
--luxury-gold: #bfa14a;
--luxury-red: #e53935;
```

### Update Store Info
**File**: `js/config.js` or `js/footer.js`
- Store name
- Owner info
- Contact details
- WhatsApp number

### Add Product Images
1. Save images to `assets/img/`
2. Update image paths in `product.js`
3. Resize to ~200x200px
4. Use PNG or JPG
